Welcome to intake-avro's documentation!
=======================================

This package enables the loading of Apache Avro files within the Intake data loading and catalog system.
Two plugins are contained: for fast loading of strictly tabular data to pandas dataframes, and slower
reading of more complicatedly structured data as a sequence of python dictionaries.
